R√©gression lin√©aire simple#

Mod√®le simple : une variable √† expliquer $Y$ et une seule variable explicative $X$.

$$y_i = \beta_0 + \beta_1 X_i + \epsilon_i$$

Hypoth√®ses √† v√©rifier pour la r√©gression lin√©aire simple :

  1. il existe une corr√©lation lin√©aire entre $X$ et $Y$

  2. la distribution de l‚Äôerreur $\epsilon$ est ind√©pendante de la variable X (exog√©n√©it√©)

  3. l‚Äôerreur suit une loi normale centr√©e i.e. $E(Œµ_i) = 0$

  4. l‚Äôerreur est de variance constante (homosc√©dasticit√©) i.e $Var(Œµ_i) = \sigma^2$, une constante

  5. les erreurs sont ind√©pendantes (absence d‚Äôautocorr√©lation) i.e. $Cov(Œµi, Œµj) = 0$, pour tout $i, j \in N \times N, i \neq j$

R√©gression lin√©aire multiple#

Mod√®le multiple : une variable √† expliquer $Y$ et $P$ variables explicatives $X_i$

$$y_i = \beta_0 + \beta_1 X_i(1) + \beta_2 X_i(2) + \beta_3 X_i(3) + ‚Ä¶ + \beta_P X_i(P) + \epsilon_i$$

Hypoth√®ses √† v√©rifier pour la r√©gression lin√©aire multiple :

  1. il existe une corr√©lation lin√©aire entre $X_i$ et $Y$

  2. $Cov(X_i, Œµ_j) = 0$, pour tout $i, j \in N \times N, i \neq j$ (exog√©n√©it√©) et pour chaque variable explicative $X_p$

  3. l‚Äôerreur suit une loi normale centr√©e i.e. $E(Œµ_i) = 0$

  4. l‚Äôerreur est de variance constante (homosc√©dasticit√©) i.e $Var(Œµ_i) = \sigma^2$, une constante

  5. les erreurs sont ind√©pendantes (absence d‚Äôautocorr√©lation) i.e. $Cov(Œµi, Œµj) = 0$, pour tout $i, j \in N \times N, i \neq j$

  6. absence de colin√©arit√© entre les variables explicatives, i.e. $X^t * X$ est r√©guli√®re, $det(X^t * X) \neq 0$

Pour les besoins de la r√©gression lin√©aire, nous cr√©ons des variables muettes (dummy variables) pour inclure les variables cat√©goriques Education et Marital_status √† la r√©gression.

R√©gression par mod√®le lin√©aire g√©n√©ralis√© (GLM) : r√©gression de Poisson#

  • Notons $Y$ notre variable d‚Äôint√©r√™t et $X_1, ‚Ä¶, X_P$ nos variables explicatives.

  • On suppose que Y suit une loi de Poisson de param√®tre $\lambda \in \mathbb{R+*}$

  • On cherche √† d√©terminer les coefficients de l‚Äô√©quation :

$\log(\mathrm{E}(Y|X)) = \log( \lambda ) = \alpha + \beta_1 X_1 + \beta_2 X_2 + ‚Ä¶ + \beta_P X_P $

Ce qui revient √† d√©terminer les coefficients de l‚Äô√©quation :

$\lambda = e^{\alpha + \beta_1 X_1 + \beta_2 X_2 + ‚Ä¶ + \beta_P X_P}$

L‚Äôobjectif de la r√©gression est d‚Äôestimer $\alpha, \beta_1, ‚Ä¶, \beta_P$. Une fois ces coefficients estim√©s, on peut d√©terminer, pour un nouveau vecteur $X$, $E(Y|X) = e^{\alpha + \beta_1 X_1 + \beta_2 X_2 + ‚Ä¶ + \beta_P X_P}$

Hypoth√®ses √† v√©rifier :#


- Avant de choisir la r√©gression de Poisson :
